5.1. Display

We can change block and inline elements property by changing display field.

5.1.1. Common Block Elements

Block elements are those taking the whole width.

  • Blocks elements are:
    • <p>
    • <h*>
    • <div>
    • <ol>, <ul>, <li>
    • <form>

5.1.2. Common Inline Elements

  • Inline elements are:
    • <span>
    • <img>
    • <a>

5.1.3. Display field

  • Block takes the whole line
  • Inline cannot change the width
  • Inline-Block default for image inline and can also change width
  • None get rid of the element

5.2. Position

  • Static default
  • Relative relative to static change position to relative, then write left/right/top/bottom change to one element of relative doesn’t affect others
  • Absolute relative to its parent
  • Fixed Fixed to the display screen

5.3. Centrialize

5.3.1. Use text-align

text-align only applies to inline-block elements or elements that take the whole width only applies to elements that doesn’t have width set

5.3.2. Use margin

margin: 0 auto 0 auto

5.4. Font

change font-family we might need some fallbacks check embedded fonts : Google Fonts

  • serif default
  • sans-serif no feet

5.5. Font size

5.5.1. Dynamic font size

use percentage or em will inherit from parent element e.g. parent font 2em, current font set to 2em, results in 4em. use rem to ignore parent font(CSS3)

5.5.2. Static font size

use px

5.6. Float and Clear

Make an element float to allow elements display on its row. Make an element clear to disallow elements display on its row.
